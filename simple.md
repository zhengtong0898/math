# 乘方
- 一次方  
  $ 2^1 $
- 二次方(也被称为平方)  
  $ 2^2 $
- 三次方(也被称为立方)  
  $ 2^3 $    

```python3
firstPowerOfTwo = 2 ** 1      # firstPowerOfTwo = pow(2, 1)       # 一次方
secondPowerOfTwo = 2 ** 2     # secondPowerOfTwo = pow(2, 2)      # 二次方(平方)
thirdPowerOfTwo = 2 ** 3      # thirdPowerOfTwo = pow(2, 3)       # 三次方(立方)
```
